# Etapas 01 – Ubuntu Server 24.04.4 LTS diegimas į Raspberry Pi 5

## 0. Greitas startas
1. Įrašyti Ubuntu per Raspberry Pi Imager
2. Įjungti SSH per Advanced Options
3. Paleisti Raspberry Pi
4. Prisijungti per SSH
5. Paleisti first_boot_prep.sh
6. Patikrinti su check_ubuntu_stage1.sh

## 1. Etapo paskirtis
Paruošti 128 GB microSD kortelę su Ubuntu Server 24.04.4 LTS (64-bit), paleisti Raspberry Pi 5 ir užtikrinti, kad sistema veiktų per tinklą bei būtų paruošta ROS 2 diegimui.

## 2. Tikslas
- Įrašyti OS
- Paleisti Pi
- Užtikrinti SSH
- Patikrinti tinklą
- Užfiksuoti etapą

## 3. Testai
- Boot veikia
- SSH veikia
- Tinklas veikia
- Sistema atnaujinta
